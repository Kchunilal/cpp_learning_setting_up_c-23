# C++ Learning Repository

My personal workspace for learning modern C++ (C++23) using Visual Studio Code on macOS.

---

### Goal
To build a strong foundation in modern C++ through daily practice, following [learncpp.com](https://www.learncpp.com/) and other resources while using professional tools and best practices.

### Current Setup
- **Compiler**: clang++ (Xcode Command Line Tools)
- **Language Standard**: C++23 (latest stable standard)
- **IDE**: Visual Studio Code
- **Build System**: Custom VS Code tasks (Cmd + Shift + B)
- **Warnings**: Enabled for better learning (`-Wall -Wextra -Wsign-conversion -Wconversion`)

---

### How to Compile & Run

1. Open any `.cpp` file
2. Press **`Cmd + Shift + B`**
3. Select **"Build & Run C++23"**

The program will compile with C++23 and run automatically.
